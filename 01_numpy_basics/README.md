# NumPy Basics Practice

This folder contains introductory practice programs for learning **NumPy**, a fundamental library used for numerical computing and data processing in Python.

## Topics Practiced

* Importing NumPy library
* Creating NumPy arrays
* Performing basic array operations
* Calculating sum, mean, maximum and minimum values
* Understanding efficient numerical computation using arrays

## Programs Included

1. **array_basics.py**
   Demonstrates creation of a NumPy array and performs basic statistical operations such as sum, mean, max and min.

## Learning Outcome

This practice helps in building the foundation required for data analysis, machine learning, and scientific computing. NumPy provides faster and more memory-efficient operations compared to traditional Python lists and is widely used in real-world data workflows.
